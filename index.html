Este Quiz foi criado para testar seus conhecimentos em programação. Responda às perguntas abaixo e descubra o quanto você sabe sobre linguagens de programação, estruturas de dados e algoritmos. Você poderá encontrar alguns acrônimos como HTML e CSS, que são comumente usados em desenvolvimento web.

OBS: Este quiz foi feito apenas para praticar conceitos basicos de HTML e CSS e não deve ser levado tão à sério as escolhas das perguntas ou sua lógica.
